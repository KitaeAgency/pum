Quick development setup
-----------------------

To quickly get a ready-to-use application, clone repository and execute bash
script:

.. code-block:: bash

    git clone git@github.com:alexandresalome/pum.git
    cd pum

    # edit app/config/parameters.yml

    ./reset.sh
